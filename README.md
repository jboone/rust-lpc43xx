# Generate HW interface code from SVD

Download from https://www.lpcware.com/content/nxpfile/lpc43sxx-svd-file

```
patch -p0 LPC43xx_43Sxx.svd LPC43xx.patch
svd2rust -i LPC43xx_43Sxx.svd | rustfmt >src/lib.rs
```
